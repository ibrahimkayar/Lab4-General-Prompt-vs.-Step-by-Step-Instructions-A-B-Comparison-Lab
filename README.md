# Lab 4 – General Prompt vs. Step-by-Step Instructions A/B Comparison

## Objective
The objective of this lab is to compare a general prompt with a step-by-step prompt and analyze how prompt structure affects the quality of AI-generated outputs.

## Task
Create a 7-day study plan for improving Python and data science skills.

## Prompt A – General Prompt

Create a 7-day study plan for improving Python and data science skills. Include daily topics, practice tasks, and expected outcomes.


## Prompt B – Step-by-Step Prompt

You are an AI study coach for a university statistics student.

Create a 7-day study plan for improving Python and data science skills.

Follow these steps:
1. First, identify the main learning goals for a beginner-to-intermediate student.
2. Then, divide the goals into 7 daily study topics.
3. For each day, provide:
   - Topic
   - Short explanation
   - Practice task
   - Expected outcome
4. Make the plan realistic for a student who can study 2 hours per day.
5. Present the final answer in a clear table format.
6. Use simple and student-friendly language.



## Comparison

Prompt A produced a general study plan. It included useful topics and tasks, but the output was less structured and harder to use immediately.

Prompt B produced a more organized and practical result. Because the prompt included clear steps, constraints, and an output format, the AI created learning goals, daily topics, practice tasks, and expected outcomes in a clearer way.

## Conclusion

Step-by-step prompting gives more reliable, structured, and usable outputs than a simple general prompt. I would use Prompt A for simple tasks, but I would use Prompt B for complex tasks that require planning, reasoning, and a clear format.
